# Laravel Sanctum API skeleton
*The Simple Laravel Sanctum API Skeleton with a simple demo Vue single page application*
#### Back-End
 + Laravel
 + Sanctum  
 
#### Front-End
 + Vue.js (Simple SPA)
 + Bootstrap

## Installation
#### Copy:  
`git clone`  
#### Install:  
`composer install`  
`npm install`  
#### Build:  
`npm run watch`
